# Samsung Supply Chain Analytics Dashboard

## Project Overview

This project presents an end-to-end Supply Chain Analytics solution for Samsung Electronics using Power BI. The dashboard provides insights into supplier performance, production efficiency, inventory management, shipment tracking, and sales performance.

## Business Problem

Supply chain operations involve multiple stakeholders, including suppliers, manufacturing plants, logistics providers, and customers. The objective of this dashboard is to monitor key operational metrics and support data-driven decision-making.
